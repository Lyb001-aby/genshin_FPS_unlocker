# genshin_FPS_unlocker
原神帧率解锁（无右ctrl键的键盘的特化）  

## 📖 简介

本项目基于 [xiaonian233/genshin-fps-unlock](https://github.com/xiaonian233/genshin-fps-unlock) 进行了键盘适配优化。原版工具使用右Ctrl键作为功能开关，但部分键盘（如小配列键盘、笔记本键盘等）没有独立的右Ctrl键，导致部分功能受限。本版本针对这一问题进行了特化改进，第一次运行会自动配置出一个txt，后续用户可直接调整txt的FPS项热修改帧率（有效值24-240FPS）

**原神帧率解锁工具**可以突破游戏默认的60帧上限，让您在高刷新率显示器上获得更流畅的游戏体验。
