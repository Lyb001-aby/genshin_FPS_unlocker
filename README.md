# genshin_FPS_unlocker
原神帧率解锁（无右ctrl键的键盘的特化）  

## 📖 简介

本项目基于 [xiaonian233/genshin-fps-unlock](https://github.com/xiaonian233/genshin-fps-unlock) 进行了键盘适配优化。原版工具使用右Ctrl键作为功能开关，但部分键盘（如小配列键盘、笔记本键盘等）没有独立的右Ctrl键，导致部分功能受限。本版本针对这一问题进行了特化改进，第一次运行会自动配置出一个txt，后续用户可直接调整txt的FPS项热修改帧率（有效值24-240FPS）

**原神帧率解锁工具**可以突破游戏默认的60帧上限，让您在高刷新率显示器上获得更流畅的游戏体验。

### 食用方法

> ⚠️ **重要：本程序必须以管理员身份运行！**  
> 右键点击 `genshinFPS.exe` → 选择「以管理员身份运行」

#### 首次使用
1. 从 [Releases](https://github.com/xiaonian233/genshin-fps-unlock/releases) 下载最新版本
2. **以管理员身份运行** `genshinFPS.exe`
3. 首次运行程序时，会提示启动原神
4. 启动原神后，程序会自动获取原神路径
5. 程序会在同目录下自动生成配置文件 `fps_config.txt`
   > 📁 推荐将 `genshinFPS.exe` 单独放在一个文件夹中，便于管理

#### 日常使用
1. 关闭游戏进程（如果正在运行）
2. **以管理员身份运行** `genshinFPS.exe`
3. 程序会自动启动原神并进行注入操作
4. 游戏运行时，**实时修改帧率**：
   - 修改 `fps_config.txt` 中的 FPS 数值
   - 保存文件即可即时生效，无需重启游戏

### ⚙️ 配置机制与保护措施

为防止一些别有用心的破坏，程序实现了以下保护机制：

| 场景 | 处理方式 |
|------|----------|
| **FPS有效值范围** | 仅接受 **24-240 FPS** 的整数 |
| **输入非法数据** | 自动设置为 **120 FPS** 默认值 |
| **检测到被异常删除配置文件** | 自动锁定为最后一次读取的 FPS，防止程序崩溃 |
| **检测到程序进程异常退出** | 自动终止原神进程，避免游戏发出 `sync failed` 错误报告 |

> 🔒 **安全说明**：当您在程序运行时故意删除 txt 配置文件，程序会自动锁定为最后一次读取的 FPS 防止崩溃。同时，为防止中途退出本程序进程导致原神进程发出 sync failed 错误报告，程序设置了进程退出时自动杀除原神进程。
